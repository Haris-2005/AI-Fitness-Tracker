# 🏋️ AI Fitness Tracker

AI Fitness Tracker is an intelligent health and fitness web application that combines fitness tracking with Generative AI to provide personalized health guidance, workout recommendations, and nutrition support.

Built with Python, Streamlit, SQLite, and Llama AI integration, this project acts as a personal AI fitness coach for users.

---

## 🚀 Features

- 🤖 AI Fitness Coach Chatbot
- 🏋️ Personalized Workout Guidance
- 💧 Water Intake Tracking
- 🥗 Nutrition Monitoring
- 📈 Progress Tracking Dashboard
- 📔 Gym Diary / Workout Logging
- 👤 User Profile Management
- 🌙 Dark / Light Theme Support
- 🧠 AI-powered health recommendations

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **SQLite**
- **Llama 3.2**
- **Generative AI**
- **Prompt Engineering**
- **Git & GitHub**
